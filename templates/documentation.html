Soccer Player Valuation and Prediction Project

This project uses various technologies to collect, transform, and analyze soccer player data, and build a web application for users to interact with the data. The following are the important technologies used:

  • API-Football.com API
  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn
  • PySpark
  • Flask
  • DynamoDB
  • Docker
  • Amazon S3
  • Amazon Kubernetes Services (AKS)

Data Pipeline

The data pipeline starts with collecting data from the API-Football.com API using a Python script with the Requests library. Since free subscriptions have limited downloads, the dataset was downloaded from Kaggle and uploaded to Amazon S3. The data was then transformed using Pandas and NumPy, and stored back in Amazon S3.

Data Visualization

The data was visualized using Matplotlib and Seaborn libraries. Various visualizations were created, such as bar charts, box plots, and line plots. A radar plot was also created using Matplotlib to visualize the average values of some player attributes like age, overall, potential, value, and wage. All files were saved on Amazon S3.

Machine Learning

PySpark was used to apply linear regression, random forest, and decision tree models to find the best fit. Further data cleaning and tuning was done to reevaluate the models.

Web Application

A Flask web application was built with a user interface hosted on AWS. The website provides information about the project, and allows users to access data visualizations. A little game was also added where users can enter a player's name and get the player's value. The data collected from the website is stored in DynamoDB.

Docker and AKS

The application was packaged with Docker to allow for easy deployment and portability. Amazon Kubernetes Services (AKS) was used to deploy, scale, and manage the containerized application.
